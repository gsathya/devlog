#tor-bots is an IRC channel where new trac changes such as creating a new ticket, changing/modifying a ticket, code push to official repo's, tor consensus information, etc are posted by a bot. It kinda got pretty hard to figure out the interesting information from the rest of the crap and I sure as hell wasn't going to subscribe to the tor-commits mailing-list that posted each of these changes as a separate email.

I ended up making a bot of my own that listens for msgs that contained "interesting words" defined by me and pinged me about it in another channel that I idle on. It's written in Python and uses Twisted(which is just awesomesauce). Torbot currently lives at https://github.com/gsathya/torbot.

Also, I finally managed to get one of my classmates to finally contribute to Open Source. It involved quite a bit of handholding and explaining in detail about git rebase, git format-patch, tabs vs spaces, etc. But we managed to get a patch out, it's currently under review and he's already looking at other bugs to fix. That was time well spent :)
